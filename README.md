# kab
kab the lab manager
 

## INSTALLATION - LINUX

```
git clone https://github.com/d4rckh/kab
cd kab
chmod +x ./install.sh
chmod +x ./setup.sh
chmod +x ./start.sh
./install.sh
./setup.sh
```

## RUNNING

**I RECOMMEND YOU USE PM2 / SCREEN / TMUX TO RUN THIS APP**

```
./start.sh
```


## INSTALLATION - WINDOWS

```
git clone https://github.com/d4rckh/kab
cd kab
./install.bat
./setup.bat
```

## RUNNING

```
./start.bat
```
